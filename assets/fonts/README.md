# Fonts

Add these files for Instrument Serif (design system display font):

- `InstrumentSerif-Regular.ttf`
- `InstrumentSerif-Italic.ttf`

Download from [Google Fonts](https://fonts.google.com/specimen/Instrument+Serif) or the design team.

Syne and Geist Mono are loaded via `@expo-google-fonts/syne` and `@expo-google-fonts/geist-mono`.
